

#  Exam Seating Arrangement System

A web-based platform developed to simplify and digitalize the exam seating process in educational institutions.  
This project eliminates the need for printed seating charts or large PDF lists on notice boards by allowing students to directly check their exam room and bench details online.  

---

# Overview

During exam seasons, students and exam departments often face confusion and delays due to manual seating arrangement processes.  
This project offers a centralized, digital, and user-friendly system where:

- **Students** can view their seating details by entering roll or seat numbers.  
- **Admins** can upload, update, and manage seating arrangements via a secure backend.  

By reducing manual effort and paper usage, this system contributes to efficiency, sustainability, and transparency in examination management.

---

# Features

# Student Interface
- Enter roll number or seat number to instantly view:
  - Exam date  
  - Room number  
  - Bench number  
  - Subject details  

# Admin Interface
- Upload seating data via Excel or CSV file  
- Modify and update records dynamically  
- Manage exam room allocations  
- Generate and view seating charts instantly  

# System Benefits
- Instant access to seating details  
- Real-time updates for schedule or room changes  
- Reduced paper waste and manual errors  
- Accessible on both mobile and desktop devices

# System Architecture

1. **Frontend:**  
   Simple and responsive UI designed using HTML and CSS.

2. **Backend:**  
   Flask framework handles routing, logic, and communication between frontend and database.

3. **Database:**  
   MySQL stores all student details, exam schedules, and seating allocations.
